# **App Name**: Gemini Switch

## Core Features:

- Real-time Chat: Chat interface for real-time messaging.
- Online AI Chat (Gemini): Use Gemini Pro API to generate responses when online.
- Offline AI Chat (Ollama): Utilize Ollama local API for response generation in offline mode.
- Intelligent Mode Switching: Automatically switches to local LLM when internet not available.
- Mode Indicator: Clear visual indication of the current active AI model.
- AI-Powered Prompt Enhancement: A tool to refine or improve the user's prompt.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to signify intelligence and connectivity.
- Background color: Light gray (#EEEEEE) for a clean, readable interface.
- Accent color: Cyan (#00BCD4) for highlights and interactive elements.
- Body and headline font: 'PT Sans', a modern, clear sans-serif, is recommended for both headlines and body text to ensure readability.
- Use flat, minimalist icons to represent different chat functions.
- Clean, spacious layout with a clear separation between messages.
